# PianoFlow

🎹 实时钢琴 MIDI 可视化播放器 — 瀑布流音符、粒子特效、88 键虚拟键盘

[在线演示](https://fdkkkkkkkk.github.io/midi-PianoFlow/)
## 功能

- 上传并播放 `.mid` 钢琴曲
- 88 键全键盘实时高亮
- 音符瀑布流下落 + 粒子特效
- 进度条拖动跳转
- 全屏模式
- 延音踏板支持

## 技术栈

[PixiJS](https://pixijs.com/) + [Tone.js](https://tonejs.github.io/) + [d-piano](https://github.com/luiguild/d-piano)

## 运行

```bash
npm install
npm run dev
```

## 构建

```bash
npm run build
```
